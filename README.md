# FastAPI Assignment 1

# Description
This project demonstrates a simple FastAPI application that returns output using an HTML template.

# Technologies Used
- Python
- FastAPI
- HTML

# How to Run
1. Install FastAPI and Uvicorn
2. Run the command:
uvicorn  main:app --reload
3. Open the browser and go to:
http://127.0.0.1:8000
